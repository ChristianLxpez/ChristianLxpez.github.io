Proyectos destacados

Extracción y automatización de datos (INHUB, 2022-2025)

En INHUB desarrollé scripts en Python usando Selenium y BeautifulSoup para extraer datos de múltiples sitios web, programando tareas diarias según el origen. Los datos se almacenaban automáticamente en Excel, CSV y bases MySQL, optimizando la toma de decisiones y reduciendo el tiempo manual.

Stack: Python, Selenium, BeautifulSoup / Task Scheduler, MySQL

App integral para mascotas (TFG Ingeniería)

Desarrollé una aplicación híbrida orientada al bienestar físico, mental y emocional de mascotas, con planes personalizados mediante IA, integrando Node.js, Express, Expo y Firebase para autenticación y almacenamiento seguro en Firestore. El sistema permite seguimiento de salud, alertas y recomendaciones dinámicas.

Stack: Node.js, Express, Expo, Firebase (Firestore, Auth), NoSQL

Plan de marketing para la app (TFG ADE)

Elaboré el plan estratégico para el lanzamiento de la app, incluyendo análisis de mercado, definición del buyer persona, estrategia digital multicanal y proyecciones financieras.
